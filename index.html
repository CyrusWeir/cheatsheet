HTML Tables

Tables are a nice, structured way to display your information - they also offer some different options in terms of CSS (borders and the like - I'd imagine they style better, but remains to be seen).

Important Tips to remember

Element

Fuction

Helpful Information

<table>

Starts a table

Helps to split up into header and body with <thead> & <tbody> functions - works with data appearing in rows - need to specify each row and then each column.

<tr>

Starts a row

essential to introduce new text/data into the table - need to use the <td> function to include data in different columns on that row.

<td>

Starts a new cell in a row

makes a new cell in the row, and as far as I know, it's the only way to do it!.

HTML Forms and Other Inputs

These are some of the inputs that we can use when setting up a HTML file

Important Tips to remember

Element

Fuction

Helpful Information

<form>

prepares the page for insertion

gotta have this one to open up the forms input context - prep with action="name.html" method="GET"

<input>

prepares the page for an input

needs to be accompanied with a 'type'. Can be radio (yes/no - single response), text, number, can also be accompanied by options (when using the select element)

<label>

preceeds the input element. Essential

Needs to include what it's for, and then can feature the text corresponding to the actual input section. The for classification should correspond with the id in the input/select element.
